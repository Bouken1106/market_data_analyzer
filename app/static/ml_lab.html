Parameter Guide（まとめ）

`Sequence Length`: 過去何日を見るか。長いほど長期文脈を拾う。

`Hidden Size` / `Layers`: モデル容量。大きいと表現力は上がるが重くなる。

`Dropout`: 過学習対策。大きすぎると学習不足になりやすい。

`Epochs` / `Patience`: 学習回数上限と早期終了条件。計算時間とのトレードオフ。

`Years`: 学習データ期間。現仕様では5年以上を前提。

設定後に実行してください。

Assumptions

各日で、その日に予測した翌営業日分布のみを使って投資比率を決定。

リスク上限: 翌日で資産の1%超損失となる確率が3%を超えない株式比率までに制限。

その上限内で期待リターン最大（0〜100%）の比率を採用。

実績リターンで日次更新して、直近60営業日の実現パフォーマンスを算出。

比較対象は、初日に買った株数を売買せず持ち続ける固定株数ホールド。

Quantile Function Plot

Plot Guide（Quantile Function）

各色の線: テスト期間の代表日ごとの分位点カーブ（1本=1日）。

横軸: 分位点 τ（0.001〜0.999）。右に行くほど「高い値側」の予測。

縦軸: 予測値（Scaleで Returns / Prices を切替）。

凡例の actual: その代表日に対する翌営業日の実測値。

テスト期間の代表日を表示します。

Fan Chart

Plot Guide（Fan Chart）

水色の線: 中央値予測（q50）。

濃い帯: 50%予測区間（q25〜q75）。

薄い帯: 90%予測区間（q05〜q95）。

黄色の点: 実測値（翌営業日の実現値）。帯内に入るほど被覆が良い。

q50（中央値）、50%帯、90%帯、実測値を表示します。
